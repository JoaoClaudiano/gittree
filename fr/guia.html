Qu'est-ce que GitTree ?

GitTree est un outil web gratuit qui transforme les dépôts GitHub en visualisations interactives, vous permettant d'explorer la structure des fichiers, d'analyser des métriques et de comprendre l'organisation du code de manière intuitive.

Because GitTree operates entirely client-side, your browser communicates directly with the GitHub API. No code is sent to or stored on any intermediate server. Analysis results are cached locally in your browser's storage to enable near-instant reloads of recently visited repositories.

Démarrer en 3 Étapes

1 Collez l'URL GitHub

Dans le champ principal de la page d'accueil, collez n'importe quelle URL de dépôt GitHub public.

Formats acceptés :
• https://github.com/usuario/repositorio
• usuario/repositorio
• https://github.com/usuario/repositorio.git

2 Cliquez sur "Visualiser"

L'outil récupérera la structure complète du dépôt et traitera toutes les informations. Le processus peut prendre quelques secondes selon la taille du dépôt.

3 Explorez les Résultats

Naviguez entre les onglets "Arbre" et "Métriques" pour explorer différentes visualisations des données du dépôt.

Fonctionnalités Principales

Vue Arborescente

Explorez la structure hiérarchique des dossiers et fichiers :

  • Cliquez sur les dossiers pour développer/réduire
  • Utilisez la recherche pour trouver des fichiers spécifiques
  • Boutons Développer/Réduire pour contrôler tout l'arbre
  • Copier la structure en texte ou Markdown
Conseil : Utilisez le bouton "Copier Texte" dans la barre d'outils pour copier la structure au format ASCII et la coller dans des documentations.

Onglet Métriques

Obtenez des informations sur le dépôt :

  • Nombre de fichiers et dossiers
  • Taille totale du dépôt
  • Graphique de distribution par type de fichier
  • Statistiques GitHub (étoiles, forks, observateurs)

Que fait chaque bouton ?

Champ de Saisie

Coller

Colle automatiquement le lien du dépôt depuis le presse-papiers. Utile quand vous venez de copier une URL GitHub.

Visualiser

Lance l'analyse du dépôt. Récupère la structure complète et traite toutes les informations sur les fichiers et dossiers.

Visualisation

Arbre

Affiche la structure du dépôt en format arborescent hiérarchique. Permet de développer et naviguer dans tous les répertoires et fichiers.

Métriques

Affiche des statistiques et graphiques sur le dépôt, y compris le nombre de fichiers, la taille totale, la distribution par type et les données GitHub.

Outils

Vider le Cache

Supprime tous les dépôts sauvegardés dans le cache local. Utilisez-le pour libérer de l'espace ou forcer une nouvelle analyse.

Recharger

Force une nouvelle analyse du dépôt actuel, en ignorant le cache. Utile pour voir les mises à jour récentes du dépôt.

Exporter

JSON

Exporte la structure complète au format JSON. Idéal pour l'analyse programmatique ou l'intégration avec d'autres outils.

CSV

Exporte la liste des fichiers au format CSV (tableur). Parfait pour l'analyse dans Excel, Google Sheets ou autres outils de données.

Barre d'Outils de l'Arbre

Copier Texte

Copie la structure arborescente au format texte ASCII. Utile pour l'inclure dans des documentations, des READMEs ou pour partager la structure.

Développer

Développe tous les répertoires de l'arbre en une fois. Affiche la structure complète du dépôt.

Réduire

Réduit tous les répertoires, n'affichant que le niveau racine. Utile pour une vue d'ensemble simplifiée.

Autres Contrôles

Basculer le Thème

Bouton dans le coin supérieur droit qui alterne entre le thème clair et sombre. Votre préférence est sauvegardée automatiquement.

Rechercher Fichiers

Champ de recherche au-dessus de l'arbre pour filtrer les fichiers en temps réel. Tapez n'importe quelle partie du nom ou du chemin du fichier.

Conseil Rapide : Passez la souris sur n'importe quel bouton pour voir une description rapide (info-bulle) de ce qu'il fait !

Conseils Avancés

Coller Rapidement : Utilisez le bouton "Coller" à côté du champ de saisie pour coller automatiquement les liens copiés depuis le presse-papiers.
Cache Intelligent : GitTree stocke les analyses localement. Les dépôts récemment analysés se chargent instantanément.
Exporter des Données : Utilisez les boutons "JSON" ou "CSV" pour exporter l'analyse complète pour une utilisation externe.
Thème Clair/Sombre : Cliquez sur l'icône lune/soleil dans le coin supérieur droit pour alterner entre les thèmes.

Questions Fréquentes

Q : GitTree fonctionne-t-il avec des dépôts privés ?
R : Non, uniquement avec des dépôts GitHub publics.

Q : Mes données sont-elles stockées ?
R : Non, le code analysé n'est pas stocké. Seules les métadonnées sont conservées localement dans votre navigateur pour le cache.

Q : Puis-je analyser n'importe quel dépôt ?
R : Oui, tout dépôt GitHub public. Certains très grands dépôts peuvent avoir leur structure tronquée par l'API GitHub.

Q : L'outil est-il gratuit ?
R : Oui, complètement gratuit et open-source.

Q: Why is the tree loading slowly?
A: Response time depends on the GitHub API and the repository size. Large monorepos with thousands of files may take several seconds. Subsequent loads of the same repository are nearly instant thanks to the local cache.

Q: How do I report a bug or suggest a feature?
A: Open an issue on the GitHub repository. All feedback is reviewed and valued.
