서비스 사용 지식재산 책임 가용성 개인정보 변경사항 연락처

GitTree에 접근하거나 사용함으로써, 귀하는 본 이용약관에 동의하는 것입니다. 여기에 기술된 조건에 동의하지 않는 경우, 해당 도구를 사용하지 않는 것을 권장합니다.

GitTree는 보조 분석 도구입니다. 인간의 검토, 기술 감사 또는 전문적 판단을 대체하지 않습니다.

1. 서비스 설명

GitTree는 소스 코드에 대한 시각화, 메트릭 및 보조 해석을 제공하기 위해 GitHub 공개 저장소의 자동 분석을 수행하는 온라인 도구입니다.

분석은 자동으로 생성되며, 분석된 코드의 품질, 보안 또는 적합성을 보증하지 않습니다.

2. 허용 가능한 사용

GitTree 사용 시 귀하는 다음에 동의합니다:

  • 공개 저장소 또는 귀하 소유 저장소의 URL만 제공
  • 합법적이고 윤리적인 목적으로만 도구 사용
  • 취약점 악용, 과부하 또는 서비스 침해 시도 금지

불법적이거나 남용적이거나 제3자의 권리를 침해하는 활동에 도구를 사용하는 것은 금지됩니다.

3. 지적 재산권

GitTree는 분석된 코드에 대한 어떠한 권리도 주장하지 않습니다.

플랫폼의 모든 독창적인 콘텐츠(인터페이스, 시각적 정체성, 텍스트 및 서비스 로직)는 저작권으로 보호되며 각 소유자에게 귀속됩니다.

서비스 사용으로 사용자에게 플랫폼의 상업적 활용에 대한 어떠한 권리도 부여되지 않습니다.

4. 책임의 제한

GitTree는 명시적 또는 묵시적 보증 없이 "있는 그대로" 제공됩니다.

다음에 대해 책임지지 않습니다:

  • 분석의 오류, 누락 또는 부정확성
  • 제시된 정보를 기반으로 한 결정
  • 도구 사용으로 인한 직접적 또는 간접적 손해

사용자는 얻은 결과의 검증, 해석 및 적용에 대해 단독으로 책임을 집니다.

5. 서비스 가용성

GitTree는 사전 통지 없이 언제든지 전체 또는 일부가 변경, 중단 또는 종료될 수 있습니다.

지속적이거나 오류 없는 가용성을 보장하지 않습니다.

6. 개인 정보 및 데이터

데이터 처리는 본 약관의 불가분의 일부인 개인정보 보호정책의 규정에 따릅니다.

사이트를 사용함으로써, 해당 정책에 명시된 바와 같이 쿠키, 분석 도구 및 광고 사용에 대해 인지하고 있음을 선언합니다.

7. 약관 변경

본 이용약관은 법적 준수 또는 서비스 발전을 위해 언제든지 수정될 수 있습니다.

변경 후 지속적인 사용은 업데이트된 버전에 대한 동의를 의미합니다.
