# Changelog

All notable changes to GitTree are documented in this file.

---

## [v1.1] — April 2026

**UI/UX Improvements** — Enhanced user experience with smarter navigation, shareable links, and improved accessibility.

### Added
- Recent repositories panel for quick access to previously analyzed repos
- GitHub API rate limit indicator with live remaining requests counter
- Keyboard shortcuts panel (press `?` to open)
- Share URL — copy a direct link to any analyzed repository
- Mobile bottom-sheet drawer for easier navigation on small screens
- Breadcrumb navigation for intuitive file path tracking

---

## [v1.0] — February 2026

**Initial Release** — First public version of GitTree, launching the core functionality for exploring GitHub repositories.

### Added
- Interactive tree visualization of any public GitHub repository
- Repository metrics and analytics (stars, forks, languages, size)
- Export as JSON, CSV, and plain text / Markdown
- Dark and light themes with automatic preference detection
- Multi-language support: English, Portuguese, Spanish, French, Italian, Japanese, Korean, Chinese
- Smart local cache for instant re-loading
- Fully responsive and accessible interface (mobile-friendly)
- Zero setup — no account, no installation, no API key required
- Open source under the MIT License
