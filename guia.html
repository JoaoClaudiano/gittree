📖 GitTree User Guide

Learn how to use all the features of the GitHub repository visualization tool

🌳 O que é o GitTree?

O GitTree é uma ferramenta web gratuita que transforma repositórios GitHub em visualizações interativas, permitindo que você explore a estrutura de arquivos, analise métricas e entenda a organização do código de forma intuitiva.

📊

Análise Visual

Veja a estrutura completa do repositório em formato de árvore

📈

Métricas Detalhadas

Estatísticas e gráficos sobre arquivos e pastas

🚀

Fácil de Usar

Basta colar a URL do GitHub e começar a explorar

🚀 Começando em 3 Passos

1 Cole a URL do GitHub

No campo principal da página inicial, cole qualquer URL de repositório público do GitHub.

Formatos aceitos:
• https://github.com/usuario/repositorio
• usuario/repositorio
• https://github.com/usuario/repositorio.git

2 Clique em "Visualizar"

A ferramenta irá buscar a estrutura completa do repositório e processar todas as informações. O processo pode levar alguns segundos dependendo do tamanho do repositório.

3 Explore os Resultados

Navegue entre as abas "Árvore" e "Métricas" para explorar diferentes visualizações dos dados do repositório.

🎯 Funcionalidades Principais

🌳 Visualização em Árvore

Explore a estrutura hierárquica de pastas e arquivos:

  • Clique em pastas para expandir/recolher
  • Use a busca para encontrar arquivos específicos
  • Botões Expandir/Recolher para controlar toda a árvore
  • Copiar estrutura como texto ou Markdown
💡 Dica: Use o botão "Copiar Texto" na toolbar para copiar a estrutura em formato ASCII e colar em documentações.

📊 Aba de Métricas

Obtenha insights sobre o repositório:

  • Contagem de arquivos e pastas
  • Tamanho total do repositório
  • Gráfico de distribuição por tipo de arquivo
  • Estatísticas de GitHub (stars, forks, watchers)

🔘 O que cada botão faz?

📥 Campo de Entrada

📋

Colar

Cola automaticamente o link do repositório da área de transferência. Útil quando você acabou de copiar uma URL do GitHub.

👁️

Visualizar

Inicia a análise do repositório. Busca a estrutura completa e processa todas as informações de arquivos e pastas.

👀 Visualização

🌳

Árvore

Mostra a estrutura do repositório em formato hierárquico de árvore. Permite expandir e navegar por todos os diretórios e arquivos.

📊

Métricas

Exibe estatísticas e gráficos sobre o repositório, incluindo contagem de arquivos, tamanho total, distribuição por tipo e dados do GitHub.

🛠️ Ferramentas

🗑️

Limpar Cache

Remove todos os repositórios salvos no cache local. Use quando quiser liberar espaço ou forçar uma nova análise.

🔄

Recarregar

Força uma nova análise do repositório atual, ignorando o cache. Útil para ver atualizações recentes no repositório.

📤 Exportar

📄

JSON

Exporta a estrutura completa em formato JSON. Ideal para análise programática ou integração com outras ferramentas.

📊

CSV

Exporta a lista de arquivos em formato CSV (planilha). Perfeito para análise em Excel, Google Sheets ou outras ferramentas de dados.

🔧 Barra de Ferramentas da Árvore

📋

Copiar Texto

Copia a estrutura da árvore em formato de texto ASCII. Útil para incluir em documentações, READMEs ou compartilhar a estrutura.

➕

Expandir

Expande todos os diretórios da árvore de uma vez. Mostra a estrutura completa do repositório.

➖

Recolher

Recolhe todos os diretórios, mostrando apenas o nível raiz. Útil para ter uma visão geral simplificada.

⚙️ Outros Controles

🌗

Alternar Tema

Botão no canto superior direito que alterna entre tema claro e escuro. Sua preferência é salva automaticamente.

🔍

Buscar Arquivos

Campo de busca acima da árvore para filtrar arquivos em tempo real. Digite qualquer parte do nome ou caminho do arquivo.

💡 Dica Rápida: Passe o mouse sobre qualquer botão para ver uma descrição rápida (tooltip) do que ele faz!

⚡ Dicas Avançadas

🔗 Colar Rapidamente: Use o botão "Colar" ao lado do campo de entrada para colar automaticamente links copiados da área de transferência.
💾 Cache Inteligente: O GitTree armazena análises localmente. Repositórios analisados recentemente carregam instantaneamente.
📤 Exportar Dados: Use os botões "JSON" ou "CSV" para exportar a análise completa para uso externo.
🌗 Tema Claro/Escuro: Clique no ícone de lua/sol no canto superior direito para alternar entre temas.

❓ Perguntas Frequentes

Q: O GitTree funciona com repositórios privados?
A: Não, apenas com repositórios públicos do GitHub.

Q: Meus dados são armazenados?
A: Não, o código analisado não é armazenado. Apenas metadados são mantidos localmente no seu navegador para cache.

Q: Posso analisar qualquer repositório?
A: Sim, qualquer repositório público do GitHub. Alguns repositórios muito grandes podem ter a estrutura truncada pela API do GitHub.

Q: A ferramenta é gratuita?
A: Sim, completamente gratuita e open-source.
