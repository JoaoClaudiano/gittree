¿Qué es GitTree?

GitTree es una herramienta web gratuita que transforma repositorios de GitHub en visualizaciones interactivas, permitiéndote explorar estructuras de archivos, analizar métricas y entender la organización del código de forma intuitiva.

Because GitTree operates entirely client-side, your browser communicates directly with the GitHub API. No code is sent to or stored on any intermediate server. Analysis results are cached locally in your browser's storage to enable near-instant reloads of recently visited repositories.

Fácil de Usar

Solo pega la URL de GitHub y empieza a explorar

Empezando en 3 Pasos

1 Pega la URL de GitHub

En el campo principal de la página de inicio, pega cualquier URL de repositorio público de GitHub.

Formatos aceptados:
• https://github.com/usuario/repositorio
• usuario/repositorio
• https://github.com/usuario/repositorio.git

2 Haz clic en "Visualizar"

La herramienta buscará la estructura completa del repositorio y procesará toda la información. El proceso puede tardar unos segundos según el tamaño del repositorio.

3 Explora los Resultados

Navega entre las pestañas "Árbol" y "Métricas" para explorar diferentes visualizaciones de los datos del repositorio.

Funcionalidades Principales

Visualización en Árbol

Explora la estructura jerárquica de carpetas y archivos:

  • Haz clic en carpetas para expandir/contraer
  • Usa la búsqueda para encontrar archivos específicos
  • Botones Expandir/Contraer para controlar todo el árbol
  • Copiar estructura como texto o Markdown
Consejo: Usa el botón "Copiar Texto" en la barra de herramientas para copiar la estructura en formato ASCII y pegarla en documentaciones.

Pestaña de Métricas

Obtén información sobre el repositorio:

  • Conteo de archivos y carpetas
  • Tamaño total del repositorio
  • Gráfico de distribución por tipo de archivo
  • Estadísticas de GitHub (stars, forks, watchers)

¿Qué hace cada botón?

Campo de Entrada

Pegar

Pega automáticamente el enlace del repositorio desde el portapapeles. Útil cuando acabas de copiar una URL de GitHub.

Visualizar

Inicia el análisis del repositorio. Obtiene la estructura completa y procesa toda la información de archivos y carpetas.

Visualización

Árbol

Muestra la estructura del repositorio en formato jerárquico de árbol. Permite expandir y navegar por todos los directorios y archivos.

Métricas

Muestra estadísticas y gráficos sobre el repositorio, incluyendo conteo de archivos, tamaño total, distribución por tipo y datos de GitHub.

Herramientas

Limpiar Caché

Elimina todos los repositorios guardados en la caché local. Úsalo cuando quieras liberar espacio o forzar un nuevo análisis.

Recargar

Fuerza un nuevo análisis del repositorio actual, ignorando la caché. Útil para ver actualizaciones recientes en el repositorio.

Exportar

JSON

Exporta la estructura completa en formato JSON. Ideal para análisis programático o integración con otras herramientas.

CSV

Exporta la lista de archivos en formato CSV (hoja de cálculo). Perfecto para análisis en Excel, Google Sheets u otras herramientas de datos.

Barra de Herramientas del Árbol

Copiar Texto

Copia la estructura del árbol en formato de texto ASCII. Útil para incluir en documentaciones, READMEs o compartir la estructura.

Expandir

Expande todos los directorios del árbol de una vez. Muestra la estructura completa del repositorio.

Contraer

Contrae todos los directorios, mostrando solo el nivel raíz. Útil para tener una visión general simplificada.

Otros Controles

Alternar Tema

Botón en la esquina superior derecha que alterna entre tema claro y oscuro. Tu preferencia se guarda automáticamente.

Buscar Archivos

Campo de búsqueda encima del árbol para filtrar archivos en tiempo real. Escribe cualquier parte del nombre o ruta del archivo.

Consejo Rápido: ¡Pasa el ratón sobre cualquier botón para ver una descripción rápida (tooltip) de lo que hace!

Consejos Avanzados

Pegar Rápido: Usa el botón "Pegar" al lado del campo de entrada para pegar automáticamente enlaces copiados del portapapeles.
Caché Inteligente: GitTree almacena análisis localmente. Los repositorios analizados recientemente se cargan al instante.
Exportar Datos: Usa los botones "JSON" o "CSV" para exportar el análisis completo para uso externo.
Tema Claro/Oscuro: Haz clic en el icono de luna/sol en la esquina superior derecha para alternar entre temas.

Preguntas Frecuentes

P: ¿GitTree funciona con repositorios privados?
R: No, solo con repositorios públicos de GitHub.

P: ¿Mis datos son almacenados?
R: No, el código analizado no se almacena. Solo los metadatos se mantienen localmente en tu navegador para el caché.

P: ¿Puedo analizar cualquier repositorio?
R: Sí, cualquier repositorio público de GitHub. Algunos repositorios muy grandes pueden tener la estructura truncada por la API de GitHub.

P: ¿La herramienta es gratuita?
R: Sí, completamente gratuita y de código abierto.

Q: Why is the tree loading slowly?
A: Response time depends on the GitHub API and the repository size. Large monorepos with thousands of files may take several seconds. Subsequent loads of the same repository are nearly instant thanks to the local cache.

Q: How do I report a bug or suggest a feature?
A: Open an issue on the GitHub repository. All feedback is reviewed and valued.
