GitTree is a free tool that allows you to visualize the complete structure of any public GitHub repository. Analyze directory trees, view file metrics, and understand code organization visually and interactively.

  • Analyze metrics and code statistics in real time
  • Explore repositories without needing to clone or download
  • Export data in multiple formats (JSON, CSV)
  • Understand complex projects intuitively
